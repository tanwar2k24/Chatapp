CUSTOM HOOK
===========

This directory hosts the deployment of a pluggable custom Hook:

- At this level, any resources needed by the Hook (e.g, configuration or properties files).
- Into the 'lib' folder, all the JAR files defining the Hook.

For more information, see Section "MQTT.Cool Hook" of the "MQTT.Cool Getting Started Guide"
(https://docs.mqtt.cool/server/guides/MQTT.Cool+Getting+Started+Guide.html#_packaging_configuration_and_deployment).
